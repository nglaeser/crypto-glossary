# Hash Functions

**Hash function** { #hash }
: A function which maps arbitrary-sized data (messages) to fixed-size strings (hash values). That is, $f: \{0,1\}^* \rightarrow \{0,1\}^n$ for some arbitrary but fixed $n$.
